# Групповое изменение системы координат   ГИС данных с помощью ArcMap

Групповая установка проекции с помощью библиотеки arcpy:
- классов пространственных объектов, поддерживаемых arcpy:
- фреймов данных mxd файла.

## Предварительные условия
Должен быть установлен и лицензирован ArcMap с библиотекой arcpy

## Инструкция по использованию
1. Скачайте скрипты на свой компьютер
2. Откройте bat-файлы, например, с помощью блокнота.
4. Добавьте в bat-файл информацию о пути к интерпретатору python, поставляемому совместно с arcgis. Для этого дублируйте строчки вида

```bash 
IF EXIST c:\python27\arcgis10.4\python.exe (
set id=10.4                           
goto :RunReplace                  
)
```
и замените номер верси **10.4** на подходящий для вас в двух местах:

- c:\python27\arcgis**10.4**\python.exe;
- set id=**10.4**.

5. Запускайте bat-файл соответствующий Вашей задаче:

- BatchDefineProjectionFeatureRun.bat -- для установки проекции классов пространственных объектов;
- BatchDefineProjectionMxdDataFrameRun.bat --  для установки проекции фреймов данных в файле mxd.

6. Следуйте инструкциям, появляющимся на экране.
